# 🎬 YouTube Downloader

Programa em Python para baixar vídeos do YouTube com escolha de qualidade (144p até 4K).

## 🚀 Como usar

1. Dê dois cliques no arquivo `rodar.bat`
2. Cole o link do vídeo do YouTube
3. Escolha a qualidade desejada
4. O vídeo será salvo na pasta `videos`

Para encerrar o programa, digite `sair`.

## 🛠️ Tecnologias utilizadas

- Python 3.14
- yt-dlp
- ffmpeg

## ⚙️ Como instalar e rodar

1. Instale o Python 3.14 em [python.org](https://python.org)
2. Instale o ffmpeg em [gyan.dev/ffmpeg/builds](https://gyan.dev/ffmpeg/builds)
3. Instale a dependência: `pip install yt-dlp`
4. Rode o programa: `py -3.14 downloader.py`
